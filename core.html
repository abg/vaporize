core — Core¶

vaporize.core.connect(user, apikey, region='DFW')¶

Create a session with the Rackspace Cloud API.

Note

Region support is not universal across all Rackspace Cloud services.

Parameters:
  • user (str) – A Rackspace Cloud username.
  • apikey (str) – A Rackspace Cloud API key.
  • region (str) – A Rackspace Cloud region, such as DFW, ORD or LON.
Raises :

ConnectionError

New in version 0.1.
