classmethod create(name, type, data, ttl=None, priority=None, comment=None)¶

Create a CloudDNS Record.

Important

This is only a factory method. In order for the Record to be created on CloudDNS you must add the Record this method returns to an existing Domain.

Parameters:
  • name (str) – A domain name such as www.mydomain.com
  • type (str) – A record type, such as A, CNAME, MX, etc.
  • data (str) – Data associated with the record (depends on type)
  • ttl (int) – Time-To-Live (TTL) in seconds
  • priority (int) – Weighted priority (required for MX records)
  • comment (str) – An optional comment or note for this record
Returns:

A Record

New in version 0.1.
