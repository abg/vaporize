classmethod create(name, flavor, size, databases, users)¶

Create a CloudDatabases Instance.

This operation asynchronously provisions a new database instance. This call requires the user to specify a flavor and a volume size. The service then provisions the instance with the requested flavor and sets up a volume of the specified size, which is the storage for the database instance.

Parameters:
  • name (str) – Name of the Instance.
  • flavor (Flavor) – The Flavor of the CloudDatabase instance.
  • size (int) – Specifies the volume size in gigabytes (GB). The value specified must be between 1 and 10.
  • databases (list of Database) – A list of Databases to create for the instance.
  • users (list of User) – A list of Users to create for the instance.
Returns:

A shiny new CloudDatabases Instance.
Return type:

Instance
